    macOS 常见问题解决办法
    • 教程
    • macOS
    • 操作
    对于刚接触 macOS 的用户来说，突如其来的各种弹窗警告会让他们措手不及，下面我就来介绍一下一些弹窗的解决措施。

    弹窗 “XXX” is damaged and can’t be opened. You should move it to the Trash

    方法一

    依次打开 System Preferences - Security & Privacy 并勾选下方的 AnyWhere 即可，如果没有 AnyWhere 选项，执行下述操作。

    打开终端并输入下列代码：

    sudo spctl --master-disable

    此时 AnyWhere 会出现，选择即可。

    提示

    如果方法一执行完后，任然出现提示，则执行方法二。

    方法二

    将需要打开的程序放入 Applications 文件夹中。

    打开终端并输入下列代码（其中 XXX 为程序名称）即可。

    xattr -cr /Applications/XXX.app
