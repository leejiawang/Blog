    Markdown 基础语法
    • 文档
    • Markdown
    • Web
    Markdown 是一种轻量级标记语言，她轻量化、易读易写特性，并且对于图片，图表、数学式都有支持，语法也极其简单易学。

    Markdown 的创始人为约翰·格鲁伯 John Gruber。它允许使用易读易写的纯文本格式编写文档，然后转换成有效的HTML文档。这种语言吸收了很多在电子邮件中已有的纯文本标记的特性1，目前许多网站都广泛使用 Markdown 来撰写说明文件或是用于论坛上发表讯息。

    标题

    输入

    ### 标题3 eyecus
#### 标题4 eyecus
##### 标题5 eyecus
###### 标题6 eyecus

    输出

    标题3 eyecus

    标题4 eyecus

    标题5 eyecus
    标题6 eyecus

    文本

    输入

    正常文本 eyecus 
**粗体 eyecus** 
*斜体 eyecus* 
~~删除线 eyecus~~
[链接 eyecus](https://eyecus.com)

    输出

    正常文本 eyecus

    粗体 eyecus

    斜体 eyecus

    删除线 eyecus

    链接 eyecus

    列表

    输入

    1. 正常文本1 eyecus
1. 正常文本2 eyecus
1. 正常文本3 eyecus

    输出

    1. 正常文本1 eyecus
    2. 正常文本2 eyecus
    3. 正常文本3 eyecus

    输入

    - 正常文本4 eyecus
- 正常文本5 eyecus
- 正常文本6 eyecus

    输出

    • 正常文本4 eyecus
    • 正常文本5 eyecus
    • 正常文本6 eyecus

    输入

    - 正常文本7 eyecus
- 正常文本8 eyecus
  - 正常文本9 eyecus
  - 正常文本10 eyecus

    输出

    • 正常文本7 eyecus
    • 正常文本8 eyecus
      • 正常文本9 eyecus
      • 正常文本10 eyecus

    图片

    输入

    ![eyecus](https://www.eyecus.com/assets/images/eyecus_logo.svg)

    输出

    引用

    输入

    > 引用文本1 eyecus

    输出

    引用文本1 eyecus

    输入

    `引用文本2 eyecus`

    输出

    引用文本2 eyecus

    表格

    输入

    | 表头1 | eyecus | 表头2 |
| :--: | :-- | --: |
| 居中文本1 | 左对齐文本1 | 右对齐文本1 |
| 居中文本2 | 左对齐文本2 | 右对齐文本2 |

    输出

    表头1 eyecus 表头2
    居中文本1 左对齐文本1 右对齐文本1
    居中文本2 左对齐文本2 右对齐文本2
