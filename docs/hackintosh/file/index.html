    工具下载

    部分用于安装黑苹果的工具

    VMware Workstation Pro

    Workstation Pro是由VMware 提供的虚拟机工具，用于在PC上搭建macOS系统环境。

    VMware 官网下载

    UniBeast

    UniBeast是由tonymacx86 提供的苹果安装U盘制作工具，一键式制作，方便易用。

    UniBeast 官网下载

    ⚠️ 注意
    使用UniBeast需要你具备macOS操作环境，且系统语言切换成English才能运行。

    Clover EFI bootloader

    Clover EFI bootloader为UEFI系统引导安装工具，是最为核心的工具之一。

    Clover EFI 下载

    Clover Configurator

    MultiBeast是由mackie100projects 提供的用于编辑Clover EFI配置文件的工具，利用图形界面大大降低了配置Clover EFI的门槛。

    Clover Configurator 下载

    ⚠️ 注意
    使用Clover Configurator需要你具备macOS操作环境。

    MultiBeast

    MultiBeast是由tonymacx86 提供的苹果驱动安装工具，方便易用。

    MultiBeast 官网下载

    Hackintool

    Hackintool是由headkaze 提供的苹果系统VRAM修补程序。

    Hackintool 下载

    部分用于优化系统的工具及文件

    NVIDIA Drivers

    用于使macOS驱动nVida显卡的驱动程序，本下载链接由tonymacx86 提供。

    驱动下载

    ⚠️ 注意
    由于在macOS 10.14系统上Apple公司已经停止了对nVdia显卡的支持，故仅提供macOS 10.13及以前系统的nVdia显卡驱动。

    部分 Kexts 文件

    用于使macOS驱动nVida显卡的驱动程序，下表包含链接由Github 提供。
    文件名 功能 描述
    AirportBrcmFixup 网卡驱动 博通无线网卡驱动
    AppleALC 声卡驱动 启用原生macOS高清音频
    ATH9KFixup 网卡驱动 Atheros网卡驱动
    BT4LEContiunityFixup 网卡驱动 开启接力，隔空投送等功能
    CodecCommander 声卡驱动 用于耳机与内置扬声器切换
    CPUFriend 电源驱动 注入CPU动态电源数据的驱动
    FakePCIID 硬件驱动 硬件ID仿冒驱动
    DiskArbitrationFixup 外设驱动 禁止“此电脑不能读取您插入的磁盘”提示
    HibernationFixup 电源驱动 修复休眠问题
    Lilu 框架注入 一些驱动的依赖
    NightShiftUnlocker 显示驱动 开启夜览功能
    NoTouchID 外设驱动 禁用触控ID支持
    RTL8111_Driver 网卡驱动 Realtek RTL8111/8168有线网卡驱动
    SATA_unsupported 外设驱动 SATA硬盘驱动
    SystemProfilerMemoryFixup 仿冒驱动 显示关于本机的内存标签
    USB_Inject 外设驱动 USB端口注入框架
    VoodooI2C 外设驱动 I2C触控板驱动
    VoodooPS2Controller 外设驱动 PS2键盘和触控板驱动
    VirtualSMC 仿冒驱动 SMC仿冒驱动，和FakeSMC二选一
    WhateverGreen 仿冒驱动 显卡注入框架，集成了很多补丁
