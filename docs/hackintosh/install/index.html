    安装指导

    注意

    本教程需要 macOS 系统环境，如果没有 macOS 设备可使用PC安装macOS虚拟机操作。另外由于电脑配置不同，安装过程可能出现各种问题，后续驱动需进行针对性安装。本教程仅供参考。

    Hackintosh 简易安装教程

    第1步 制作系统安装盘

    • 通过AppStore下载 macOS 官方镜像文件，或者从其他途径下载 Mojave.app 并拖入应用程序文件夹

    • 使用磁盘管理工具格式化U盘为GUID分区格式Mac OS Extended（日志式），单击抹掉U盘

    • 根据自己的系统参照下述方法进系统安装U盘的制作

      • 直接使用 UniBeast 开始制作，引导过程根据设备情况选择，默认选择如下图所示

      • 使用终端制作系统安装U盘，以安装 macOS Mojave 系统为例，输入如下代码并等待制作完成

        sudo /Applications/Install\ macOS\ Mojave.app/Contents/Resources/createinstallmedia --volume /Volumes/Mojave --applicationpath /Applications/Install\ macOS\ Mojave.app --nointeraction
        注意
        根据自己实际情况修改代码内容，此代码适用于系统版本为 Mojave ，U盘名称为 Mojave。

    第2步 安装原始系统引导

    提示

    如果是通过 UniBeast 制作的系统安装U盘可跳过此步骤，因为它已经配置好了基本的系统引导。

    • 使用macOS操作环境下载并安装 Clover EFI bootloader 程序，注意安装时选择安装到系统安装U盘

    第3步 安装系统

    • 将制作好的系统安装U盘插入待安装的设备，重启进入BIOS并调整下表中的选项（如果有）

      提示

      进入BIOS台式机可使用按键Del,笔记本一般使用 F2 F12 F11 Esc（具体可查寻设备说明）

      选项 开关
      VT-d Disable
      CFG-Lock Disable
      Secure Boot Mode Disable
      IO Serial Port Disable
      XHCI Handoff Enabled

    • 选择系统安装U盘为启动首选项，按F10保存设置并重启设备，进入clover界面后，选择系统安装U盘

    • 系统安装完后进行简单地配置之后就能进入桌面了，建议首次安装配置过程中设备不要联网

    第4步 详细配置系统引导

    • 可使用 Clover Configurator 配置自己的 Clover 配置文件

    第5步 系统优化 (安装驱动)

    • 可使用MultiBeast 安装基本驱动，但这并不能保证能完全装好，需要根据出现的问题上网查询解决办法，这里推荐几个网站社区：Tonymacx86、黑苹果屋 、 黑果小兵的部落阁、 黑苹果乐园

    • 英伟达NVIDIA显卡 驱动下载

    • 下表为部分用于优化macOS的驱动程序，根据需要选择下载

    文件名 功能 描述
    AirportBrcmFixup 网卡驱动 博通无线网卡驱动
    AppleALC 声卡驱动 启用原生macOS高清音频
    ATH9KFixup 网卡驱动 Atheros网卡驱动
    BT4LEContiunityFixup 网卡驱动 开启接力，隔空投送等功能
    CodecCommander 声卡驱动 用于耳机与内置扬声器切换
    CPUFriend 电源驱动 注入CPU动态电源数据的驱动
    FakePCIID 硬件驱动 硬件ID仿冒驱动
    DiskArbitrationFixup 外设驱动 禁止“此电脑不能读取您插入的磁盘”提示
    HibernationFixup 电源驱动 修复休眠问题
    Lilu 框架注入 一些驱动的依赖
    NightShiftUnlocker 显示驱动 开启夜览功能
    NoTouchID 外设驱动 禁用触控ID支持
    RTL8111_Driver 网卡驱动 Realtek RTL8111/8168有线网卡驱动
    SATA_unsupported 外设驱动 SATA硬盘驱动
    SystemProfilerMemoryFixup 仿冒驱动 显示关于本机的内存标签
    USB_Inject 外设驱动 USB端口注入框架
    VoodooI2C 外设驱动 I2C触控板驱动
    VoodooPS2Controller 外设驱动 PS2键盘和触控板驱动
    VirtualSMC 仿冒驱动 SMC仿冒驱动，和FakeSMC二选一
    WhateverGreen 仿冒驱动 显卡注入框架，集成了很多补丁
