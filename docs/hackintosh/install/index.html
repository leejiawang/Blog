    Hackintosh 简易安装教程

    第1步 制作macOS系统安装盘

    • 首先根据自己的系统参照下述方法进系统安装U盘的制作。
      • 通过AppStore下载macOS官方镜像文件
      • 使用磁盘管理工具格式化U盘为GUID分区格式Mac OS Extended（日志式）
      • 直接使用UniBeast开始制作
      • 使用终端制作系统安装U盘
