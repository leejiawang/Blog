    HTML5 学习笔记

    HTML称为超文本标记语言，是一种标识性的语言。它包括一系列标签，通过这些标签可以将网络上的文档格式统一，使分散的Internet资源连接为一个逻辑整体。

    简单介绍

    Html简单实例 

    <!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>eyecus</title>
  </head>

  <body>
    <h1>hello world！</h1>
    <p>hello eyecus！</p>
  </body>
</html>

    hello world！

    hello eyecus！

    <!DOCTYPE html> ：声明为html5文档
    <html>：html页面的根元素
    <head>：包含了文档的元(meta)数据
    <title>：包含了页面标题
    <body>：包含了可见的页面内容
    <h1>：一级标题标签
    <p>：段落标签

    注意 对于中文网页需要使用 <meta charset="utf-8"> 声明编码，否则会出现乱码。
